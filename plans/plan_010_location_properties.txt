All locations on the map have properties.

Examples of properties are:
- soil fertility
- groundwater availability
- ore density
- cloud cover
- wind speed

These properties may influence a building's output.

Each location has a set value for all properties and that value is a floating point number.

#1 Sektors are created with a tool
- tool for creating sektors
    - tool located in frontend/src/creator folder
    - the main part is a createSektor function in createSektor.ts
    - createSektor function returns SektorData

#2 Locations have properties
- SektorData contains a location matrix: locations[][]: Location
    - Location object has property "properties" which is an object with string keys & number values
    - initially, creator generates these properties:
        - soil
        - groundwater
        - ore
        - insolation
        - wind
